pandas方法
发布于 2019-03-28   更新于 2019-07-28

导入pandas库，一般都会用到numpy库

使用iloc按位置区域提取数据

df_inner.iloc[:3,:2] #冒号前后的数字不再是索引的标签名称，而是数据所在的位置，从0开始，前三行，前两列。

适应iloc按位置单独提起数据

df_inner.iloc[[0,2,5],[4,5]] #提取第0、2、5行，4、5列

导入CSV等文件：

维度查看：

df.shape

数据表基本信息（维度、列名称、数据格式、所占空间等）：

df.info()

每一列数据的格式：

df.dtypes

某一列格式：

df[‘B’].dtype

空值：

df.isnull()

查看某一列空值：

df.isnull()

查看某一列的唯一值：

df[‘B’].unique()

查看数据表的值：

df.values

查看列名称：

df.columns

1查看前10行数据、后10行数据：

用数字0填充空值：

df.fillna(value=0)

使用列prince的均值对NA进行填充：

df[‘prince’].fillna(df[‘prince’].mean())

清楚city字段的字符空格：

df[‘city’]=df[‘city’].map(str.strip)

大小写转换：

df[‘city’]=df[‘city’].str.lower()

更改数据格式：

df[‘price’].astype(‘int’)

更改列名称：

df.rename(columns={‘category’: ‘category-size’})

删除后出现的重复值：

df[‘city’].drop_duplicates()

删除先出现的重复值：

df[‘city’].drop_duplicates(keep=’last’)

数据替换：

df[‘city’].replace(‘sh’, ‘shanghai’)

使用“与”进行筛选

使用“或”进行筛选
